REST API v3 Resources¶

Alerts¶

Alerts are received and processed before being added to database.

Resource Description
GET /api/alerts Returns all alerts or all alerts that meet the query filter.
POST /api/alert foo
GET /api/alert/<id> foo
POST /api/alert/<id>/tag foo
POST /api/alert/<id>/status foo
DELETE /api/alert/<id> foo

Environments & Services¶

Resource Description
get-environments Returns list of environments and all services for each
get-services Returns list of all services and count of alerts for each

Heartbeats¶

Heartbeats are sent from something at regular intervals to ensure that it is operating.

Resource Description
GET /api/heartbeats foo
POST /api/heartbeat foo
DELETE /api/heartbeat/<id> foo
