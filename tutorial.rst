Tutorial
========

This tutorial is going to assume that you want to deploy a typical Alerta installation that includes:

* a single web server with co-located MongoDB database
* authentication using Basic Auth with admin and user logins
* customer-specific views

* integration with Pingdom and some tags and custom attributes
